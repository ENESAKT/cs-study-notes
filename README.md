# cs-study-notes

A live experiment in structured self-study. Every course in this repo runs through the same
three-file loop, driven by an AI mentor that **never gives answers — only direction**.

## The Loop

| File | Role |
|---|---|
| `study-plan.md` | The contract: sources, order, what "done" means for the course |
| `notebook.md` | The active working memory — short notes captured while studying |
| `lesson.md` | The distilled output — structured explanations written in my own words |

The rules the AI mentor follows (answer language, file discipline, how sessions run) live in
[`methodology-agent-spec.md`](methodology-agent-spec.md). The reusable version of that mentor
system is published separately at [ai-dev-mentor](https://github.com/ENESAKT/ai-dev-mentor).

## Courses

- [`computer-networks/`](computer-networks/) — protocols, VLANs, Packet Tracer labs
- [`microprocessors/`](microprocessors/) — 8051 microcontroller
- [`system-programming/`](system-programming/) — OS and system-level programming
- [`software-engineering/`](software-engineering/) — design patterns, UML, OOP
- [`internet-technologies/`](internet-technologies/) — JavaScript, DOM, web fundamentals

Note content is in Turkish — the method is the point here, not the language.
